# ✅ Todo List App

A clean, minimal todo list app built with vanilla HTML, CSS, and JavaScript.

## Features

- Add tasks with a single click
- Mark tasks as complete with a strikethrough
- Delete individual tasks
- Live task counter that updates in real time
- Empty state message when no tasks exist
